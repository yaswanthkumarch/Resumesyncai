3. SDLC (Software Development Life Cycle)

Step-by-step development roadmap for the Sophyra Platform.

Phase 1: Planning

Requirements Analysis & Stack Selection

  • Define User Roles (HR, Candidate, Admin).
  • Select Supabase for Auth/Database.
  • Select Ollama for on-premise LLM.
  • Design Multi-tenant RBAC strategy.
Phase 2: Design

Architecture & UI/UX

  • Create Database Schema (ER Diagram).
  • Design Interview Interface (Bento grid style).
  • Design API Contracts (RESTful endpoints).
  • Define "Ticket Logic" threshold rules.
Phase 3: Implementation

Core Development

  • Backend: Node.js API + Ollama integration.
  • Database: Supabase setup + RLS Policies.
  • Frontend: React Vite app, Job Creation Flow.
  • AI Logic: Prompt engineering for scoring.
Phase 4: Testing

QA & Validation

  • Unit testing for scoring algorithms.
  • E2E testing for Interview flow.
  • Security Audit (RLS verification).
  • Load testing for concurrent interviews.
Phase 5: Deployment

Docker & CI/CD

  • Containerize App, Backend, and Ollama.
  • Configure Nginx Reverse Proxy.
  • Setup Single VM deployment script.
  • Configure SMTP for Email notifications.
